Simulator¶

The Simulator class wraps up the configuration and simulator into a class, hiding the raw FFI calls.

The code for this class can be found in /ctoybox/ctoybox/ffi.py.

class ctoybox.Simulator(game_name, sim=None)¶

The Simulator is an instance of a game configuration. You can call new_game on it to begin.

Parameters

  • game_name – one of “breakout”, “amidar”, etc.

  • sim – optionally a Rust pointer to an existing simulator.
